QuickSearchExt

QuickSearchExt is a Safari extension that searches the selected text on your favorite sites. On iPhone, jump from the popup; on Mac, from the toolbar or the right‑click menu — one tap to search.

← All apps Contact support

Quick start

How to use on iPhone / iPad

  1. Open Safari.
  2. Tap … → Extensions → launch QuickSearchExt.
  3. Tap the site you want to search.

Tip: the query auto-fills with selected text. If nothing is selected, it keeps your previous query.

How to use on Mac

  1. Safari Settings → Extensions → enable QuickSearchExt.
  2. Click the toolbar button, or right-click selected text → QuickSearchExt.

You can run the same search sites list on both platforms.

Features

  • Auto-fill with selected text; if nothing is selected, keep the previous query.
  • Built-in sites (examples): Search, Images, Maps, Video, Shopping, Encyclopedia, and more.
  • Locale-aware suggestions (enable/disable per site).
  • Reorder, set defaults, and add / edit / delete sites.
  • Clear list with site icons.
  • Multilingual localization (including Japanese).

Customization

  • Freely add a site name and a URL template (use %s for the query).
  • Reorder / enable / disable with one click.

Privacy

No personal data is collected. Searches are launched directly from the extension on your device.
